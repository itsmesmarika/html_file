steps for following for the first time after initializing the new project folder

1. make the project file 2. initialize that file git init 3. make the changes in the file , like i am making these changes 4. add the specific file or whole file inside this folder git add index.html or git add . 5. git commit -m "all the changes are going to the remote repo" 6. add the remote place where you want to push the code git remote add origin https://github.com/itsmesmarika/html_file.git 7. git push origin main 8. check if the main is branch or not git branch 9. git branch -m main 10 . if anything happen and blocking yu to push then you can forcefully push the data git push origin main --force
